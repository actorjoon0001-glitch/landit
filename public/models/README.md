# 3D 모델 (GLB)

홈플래너에서 내보낸 주택 모델(GLB)을 이 폴더에 넣습니다.

## 사용법
1. GLB 파일을 여기에 배치 (예: `nordic-24.glb`)
2. `lib/data.ts`의 해당 주택에 경로 지정:
   ```ts
   { id: "m-nordic-24", ..., model: "/models/nordic-24.glb" }
   ```
3. 시뮬레이션이 절차적 지오메트리 대신 이 모델을 기초 위에 자동 배치합니다.
   (모델이 없으면 기존 절차적 버전으로 폴백)

## 권장 내보내기
- **GLB(바이너리 glTF)** 단일 파일, 텍스처 포함
- 텍스처 ≤ 2K, meshopt/Draco 압축으로 파일당 수 MB 이내
- Y-up, 미터 단위 권장 (자동 정규화되지만 방향은 유지됨)
