# LANDIT 🌱

**Land + Build it + Land it(안착하다)**

전원주택·세컨하우스·모듈러주택을 원하는 사람이 매물로 나온 땅을 지도에서 고르면,
그 땅이 개발되었을 때의 모습과 모듈러주택이 실제로 앉혀진 모습을 **가상 시뮬레이션**으로
미리 보여주고, 마음에 들면 **토지 거래와 시공사 연결**까지 한 번에 이어주는
토지-주택 통합 플랫폼입니다.

## 이 프로토타입이 보여주는 것

| 흐름 | 화면 | 설명 |
| --- | --- | --- |
| 1. 땅을 고른다 | `/explore` | 지도에서 매물 토지를 지역별로 탐색하고 조건을 확인 |
| 2. 개발 후를 본다 | `/land/[id]` | 나대지 ↔ 개발 후 비교, 모듈러주택을 대지에 앉힌 시뮬레이션 |
| 3. 비용을 계산한다 | `/land/[id]` | 대지가 + 시공비 합산 예상 총 비용 |
| 4. 거래·시공을 잇는다 | `/land/[id]` | 상담 신청 → 토지 거래 + 시공사 연결 |

## 기술 스택

- **Next.js 15** (App Router) + **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- 지도·시뮬레이션은 외부 API 키 없이 동작하도록 **자체 SVG 렌더링**으로 구현
  (실서비스에서는 VWorld/국토부 API·3D 렌더 엔진으로 대체 가능)

## 실행

```bash
npm install
npm run dev      # http://localhost:3000
```

프로덕션 빌드:

```bash
npm run build && npm start
```

## 구조

```
app/
  page.tsx              # 랜딩
  explore/page.tsx      # 매물 지도
  land/[id]/page.tsx    # 토지 상세 + 시뮬레이션
  how-it-works/page.tsx # 이용 방법
components/
  LandMap.tsx           # 인터랙티브 지도(클라이언트)
  LandDetail.tsx        # 상세·주택선택·비용·CTA(클라이언트)
  Simulation.tsx        # 개발/주택 배치 SVG 시뮬레이션(클라이언트)
lib/
  data.ts               # 매물 토지·모듈러주택 목업 데이터
```
